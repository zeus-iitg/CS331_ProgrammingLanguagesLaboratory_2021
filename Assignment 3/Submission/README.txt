Instructions for Running the Code
---------------------------------

For Linux:
1. Make sure GHC (Glasgow Haskell Compiler for Haskell) is installed on your machine.
2. Open the terminal and navigate to the directory containing the source code.
3. For compilation, type 'ghc <filename>.hs' and press enter. 
4. To run the file, type './<filename>' and press enter.
5. You will be prompted to enter an input.
6. Provide a valid input and then press enter.
7. The output will be displayed on the screen.

For Windows:
1. Make sure GHC (Glasgow Haskell Compiler for Haskell) is installed on your machine.
2. Open the command prompt and navigate to the directory containing the source code.
3. For compilation, type 'ghc .\<filename>.hs' and press enter. 
4. To run the file, type '.\<filename>.exe' and press enter.
5. You will be prompted to enter an input.
6. Provide a valid input and then press enter.
7. The output will be displayed on the screen.

NOTE
----

Appropriate error handling has been done for all the three files. On entering an invalid input, an error message will be displayed to the user.
